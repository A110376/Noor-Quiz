# 🌙 Noor Quiz App

A beautiful and interactive Islamic quiz application built with **Next.js**.
Test your knowledge, track your score, and enjoy a clean, modern result experience.

---

## ✨ Features

* 🎯 Multiple quiz subjects (e.g. Allah Names, Seerat)
* 📊 Animated result screen with score counter
* 🎉 Confetti celebration on good performance
* 💾 LocalStorage-based result tracking
* 🎨 Modern UI with glassmorphism design
* 📱 Fully responsive layout

---

## 🛠️ Tech Stack

* **Next.js**
* **React**
* **Tailwind CSS**
* **React Confetti**
* **React Use**

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/A110376/Noor-Quiz.git
cd noor-quiz
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

Open in browser:

```
http://localhost:3000
```

---

## 📦 Build for Production

```bash
npm run build
npm start
```
